📋 HOW TO RUN THE APP (3 EASY STEPS)
═══════════════════════════════════════════════════════════════════

STEP 1: Open PowerShell in the project folder
   • Right-click on folder "translation-assistant"
   • Click "Open in Terminal" or "Open PowerShell window here"

STEP 2: Run these commands (copy-paste each line):

   python main.py --lang es

STEP 3: You should see this:
   
   🌐 TRANSLATION ASSISTANT RUNNING
   Target Language: ES
   Hotkey: Ctrl+Shift+T
   
   ✓ If you see this, the app is running!
   ✓ Keep this window OPEN (minimize it if you want)

🎯 HOW TO TRANSLATE TEXT
═══════════════════════════════════════════════════════════════════

1. Open Notepad (or any app - browser, Word, etc.)

2. Type some text:
   "Hello, how are you?"

3. SELECT the text (drag mouse over it to highlight)

4. Press Ctrl+Shift+T
   (Hold Ctrl, hold Shift, press T)

5. A POPUP WINDOW will appear showing:
   
   ┌──────────────────────────────────┐
   │  From English to Spanish         │
   │  ────────────────────────────    │
   │  Original: Hello, how are you?   │
   │  Spanish: ¿Hola, cómo estás?    │
   │                                   │
   │   [Copy] [Speak] [Save]          │
   └──────────────────────────────────┘

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

PROBLEM: "PowerShell shows an error about execution policy"

SOLUTION: Run this command first:
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Then try activating venv again.

───────────────────────────────────────────────────────────────────

PROBLEM: "App starts but closes immediately"

SOLUTION: Don't run it with "python main.py" directly.
Use this command instead:
   
   .\venv\Scripts\Activate.ps1; python main.py --lang es

Make sure virtual environment is activated (you'll see "(venv)" in prompt).

───────────────────────────────────────────────────────────────────

PROBLEM: "Popup appears but is empty"

SOLUTION: 
   • Check internet connection (needs Google Translate API)
   • Try shorter text first ("Hello")
   • Make sure text was actually selected

───────────────────────────────────────────────────────────────────

PROBLEM: "Can't press Ctrl+Shift+T, something else happens"

SOLUTION: Another app is using that hotkey.
Close browser tabs (browsers use Ctrl+Shift+T to reopen tabs).

───────────────────────────────────────────────────────────────────

PROBLEM: "Error: No module named 'something'"

SOLUTION: Run installation verification:
   python test_installation.py

Should show: ✓ Passed: 16
If any failures, reinstall packages:
   pip install -r requirements.txt
